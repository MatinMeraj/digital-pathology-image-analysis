Independent image analysis project focused on learning reproducible computational workflows for digital pathology-style images.

Goal:
Build a small Python workflow that loads tissue/cell images, applies preprocessing, segments visible structures, extracts basic quantitative features, and documents the results clearly.

Current scope:
- Load sample microscopy or tissue images
- Convert images to grayscale
- Apply thresholding and contour detection
- Measure object count, area, and intensity-related features
- Save outputs and summarize results in a reproducible notebook
